# SpringBoot网上商城

## 一、系统介绍

前台功能
商品分类查询，加入购物车，提交订单，查看订单

后台功能：
用户管理，订单管理，分类管理，商品管理

## 二、所用技术

后端技术栈：

- springboot
- jpa
- mysql

前端技术栈：

- thymeleaf


## 三、环境介绍

基础环境 :IDEA/eclipse, JDK 1.8, Mysql5.7及以上,Maven

源码+数据库脚本
